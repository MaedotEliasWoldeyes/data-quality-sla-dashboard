# Data Quality & SLA Monitoring Dashboard

Interactive Tableau dashboard designed to monitor operational performance, SLA compliance, upload failures, and transaction quality metrics.

## Tools Used

* Tableau
* SQL
* Data Visualization
* KPI Reporting
## Key Features

* SLA compliance tracking
* Upload status monitoring
* Transaction quality analysis
* Missing ID detection
* Executive KPI reporting
